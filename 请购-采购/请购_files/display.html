报表查看器配置错误

报表查看器 Web 控件 HTTP 处理程序尚未在应用程序的 web.config 文件中注册。请将 <add verb="*" path="Reserved.ReportViewerWebControl.axd" type = "Microsoft.Reporting.WebForms.HttpHandler, Microsoft.ReportViewer.WebForms, Version=14.0.0.0, Culture=neutral, PublicKeyToken=89845dcd8080cc91" /> 添加到 web.config 文件的 system.web/httpHandlers 节，或者将 <add name="ReportViewerWebControlHandler" preCondition="integratedMode" verb="*" path="Reserved.ReportViewerWebControl.axd" type="Microsoft.Reporting.WebForms.HttpHandler, Microsoft.ReportViewer.WebForms, Version=14.0.0.0, Culture=neutral, PublicKeyToken=89845dcd8080cc91" /> 添加到 Internet Information Services 7 或更高版本的 system.webServer/handlers 节。

不支持此浏览器

报表查看器需要标准的兼容浏览器。Internet Explorer 8 和较早版本的用户应切换到新式浏览器，或者如果已在使用新式浏览器，则通过将 <meta http-equiv="X-UA-Compatible" content="IE=edge" > 头标记添加到主机页来启用标准模式。
